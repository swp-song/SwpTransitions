### SwpTransitions

> * 常用转场动画的封装，无需在写麻烦的转场动画。

-------


### 导入

> * 手动导入：
> 
>> * **SwpTransitions** 文件夹导入项目中。
>> * **`#import "SwpTransitionsHeader.h`**
>> 
> -------

> * CocoaPods 导入:
> 
>> * **pod search SwpTransitions**
>> * **pod 'SwpTransitions'**
>> * **`#import <SwpTransitions/SwpTransitionsHeader.h>`**
>> 
> -------
